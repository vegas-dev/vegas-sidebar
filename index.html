Vegas Sidebar

App for creating custom sidebars, is written in JQuery, html5 and ССS3 using the compiler SCSS

  • home page
  • github

Demo

open left open right open top open bottom

Install

npm i vgsidebar

or download here

Usage

HTML markup
Panel call

<a href="#" data-toggle="vg-sidebar" data-target="#left-panel">Open panel</a>
Panel

<div class="vg-sidebar left" id="left-panel">
	<div class="vg-sidebar-content">
		<!-- Here is your content -->
	</div>
</div>

Parameter [data-target="#left-panel"] or [href="#left-panel"] indicate the path to the panel to open, this is a required parameter.

Classes left, right, top, bottom indicate the direction of opening the panel.

Init

Automatic initialization through [data-toggle="vg-sidebar"]
